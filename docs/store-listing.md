# Chrome Web Store Listing — Quick Filter for Online Sellers

## Extension Name
Quick Filter for Online Sellers (TikTok + Shopee)

## Short Description (max 132 chars)
Bulk-print 1,000+ shipping labels on TikTok & Shopee Seller Centre without pagination limits. Filter, group, and export PDFs instantly.

---

## Full Description (English)

**Print 1,000+ shipping labels in one click — no more pagination headaches.**

Quick Filter is a floating widget that embeds directly into TikTok Seller Centre and Shopee Seller Centre (Thailand). It solves the #1 frustration for high-volume sellers: TikTok clears your label selection every time you paginate, forcing you to reselect orders manually.

### Key Features

**Bulk Label Printing**
- Scan and group all pending orders across all pages instantly
- Print up to 1,000+ labels per session without losing selection
- Auto-split into chunks (e.g. 200 labels/file) for easier handling
- Parallel PDF generation — all chunks build simultaneously

**Smart Grouping & Filtering**
- Group orders by product, SKU, carrier, and quantity
- Filter by label status: Not Printed / Printed / All
- Filter by carrier (J&T, Flash, Kerry, SPX, Thailand Post, etc.)
- Pre-order filter to separate standard vs pre-order items

**PDF Alias Overlay**
- Overlay custom product nicknames directly onto shipping labels
- Per-product and per-variant alias support
- Semi-transparent watermark — doesn't obscure barcodes or QR codes

**Planning Mode**
- Assign orders to workers/team members via drag-and-drop
- Print all workers' labels in parallel with one click
- Auto-split by SKU or equal distribution
- Session persists across page refreshes

**Print History**
- Full history of every print session (up to 200 entries, 30 days)
- Re-download any previous batch without rescanning
- Search by Order ID

**Picking List PDF**
- Optional warehouse picking list generated alongside shipping labels

### Supported Platforms
- ✅ TikTok Seller Centre Thailand (seller-th.tiktok.com)
- ✅ Shopee Seller Centre Thailand (seller.shopee.co.th) — scan & group features

### Privacy
This extension does not collect, store, or transmit any personal data to external servers. All data (aliases, print history, plans) is stored locally in your browser's localStorage. Network requests are made only to TikTok and Shopee's own APIs.

## Full Description (ภาษาไทย)

**พิมพ์ฉลากขนส่ง 1,000+ ใบด้วยคลิกเดียว — ไม่ต้องเลือกซ้ำทุกหน้า**

Quick Filter คือ floating widget ที่ฝังตัวใน TikTok Seller Centre และ Shopee Seller Centre (ไทย) โดยตรง แก้ปัญหาหลักของร้านค้าที่มีออเดอร์จำนวนมาก: TikTok จะล้างการเลือกฉลากทุกครั้งที่เปลี่ยนหน้า ทำให้ต้องเสียเวลาเลือกใหม่ทุกครั้ง

### ฟีเจอร์หลัก

**พิมพ์ฉลากจำนวนมาก**
- สแกนและรวบรวมออเดอร์ที่รอพิมพ์ทุกหน้าได้ในครั้งเดียว
- พิมพ์ 1,000+ ใบต่อ session โดยไม่ต้องเลือกใหม่
- แบ่ง chunk อัตโนมัติ (เช่น 200 ใบ/ไฟล์) เพื่อความสะดวก
- สร้าง PDF ทุก chunk พร้อมกัน (Parallel) — รวดเร็ว

**กรองและจัดกลุ่ม**
- จัดกลุ่มตามสินค้า, SKU, ขนส่ง และจำนวน
- กรองตามสถานะฉลาก: ยังไม่พิมพ์ / พิมพ์แล้ว / ทั้งหมด
- กรองตามขนส่ง (J&T, Flash, Kerry, SPX, ไปรษณีย์ ฯลฯ)
- กรอง Pre-order แยกจากสินค้าปกติ

**Alias Overlay บน PDF**
- แปะชื่อย่อสินค้าลงบนฉลากขนส่งโดยตรง
- กำหนด alias ได้ทั้งระดับสินค้าและ variant
- ตัวอักษรโปร่งใส ไม่บังบาร์โค้ดหรือ QR

**Planning Mode**
- ลาก-วางออเดอร์ให้คนแพ็คแต่ละคนผ่าน drag-and-drop
- กด "พิมพ์ทุกคน" ครั้งเดียว — สร้าง PDF ทุกคนพร้อมกัน
- แบ่งงานอัตโนมัติแบบเท่ากัน หรือแบ่งตาม SKU
- Session บันทึกอัตโนมัติ

**ประวัติการพิมพ์**
- เก็บประวัติทุก session สูงสุด 200 รายการ (30 วัน)
- ดาวน์โหลดซ้ำได้โดยไม่ต้องสแกนใหม่
- ค้นหาด้วย Order ID

### แพลตฟอร์มที่รองรับ
- ✅ TikTok Seller Centre ไทย
- ✅ Shopee Seller Centre ไทย (สแกน + จัดกลุ่ม)

### ความเป็นส่วนตัว
Extension นี้ไม่เก็บหรือส่งข้อมูลส่วนบุคคลไปยังเซิร์ฟเวอร์ภายนอก ข้อมูลทั้งหมด (alias, ประวัติการพิมพ์, แผนงาน) เก็บไว้ใน localStorage ในเบราว์เซอร์ของคุณเท่านั้น
